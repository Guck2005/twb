Exercice 1 — Questions de cours 6 pts

1
Différence StatelessWidget / StatefulWidget ?

StatelessWidget est immuable : une fois construit, il ne change pas. StatefulWidget possède un objet State qui permet de modifier l'interface dynamiquement via setState().

2
Rôle de build() ?

La méthode build(BuildContext context) décrit l'interface utilisateur à afficher. Elle est appelée chaque fois que le widget doit être redessiné.

3
Rôle de Navigator.push() ?

Navigator.push() empile une nouvelle route (page) sur la pile de navigation et y navigue. L'utilisateur peut revenir avec Navigator.pop().

4
setState() dans un StatelessWidget ?

C'est impossible. Un StatelessWidget n'a pas d'objet State associé. Appeler setState() provoque une erreur de compilation car la méthode n'existe pas dans ce contexte.

5
Rôle de pubspec.yaml ?

Fichier de configuration du projet Flutter : déclare les dépendances (packages), les assets (images, polices), le nom du projet, la version SDK, et les métadonnées.

6
Deux avantages de Flutter ?

1) Code unique multi-plateforme : un seul code Dart pour iOS, Android, Web et Desktop.
2) Hot Reload : mise à jour instantanée de l'UI pendant le développement sans perdre l'état.

Exercice 2 — QCM Dart 3 pts

Programme Dart analysé : List<int> notes = [12, 8, 15, 9, 10];

1
Type de la variable notes ?

Réponse : B) List — C'est une liste d'entiers (List<int>).

2
notes.length représente ?

Réponse : B) Le nombre d'éléments dans la liste — Ici 5 éléments.

3
Combien de fois la boucle for ?

Réponse : C) 5 fois — La boucle itère sur chaque élément de la liste (5 éléments).

4
Valeur affichée pour Max ?

Réponse : B) 15 — Le maximum parmi [12, 8, 15, 9, 10].

5
Valeur affichée pour Min ?

Réponse : A) 8 — Le minimum parmi [12, 8, 15, 9, 10].

6
Nombre d'étudiants admis (note ≥ 10) ?

Réponse : B) 3 — Les notes 12, 15 et 10 sont ≥ 10.

Exercice 4 — Flutter 3 pts

1
Rôle de MaterialApp (0,5 pt) ?

MaterialApp est le widget racine d'une application Material Design. Il configure le thème, les routes, la navigation, le titre de l'app et la page d'accueil (home).

2
Rôle de Column (0,5 pt) ?

Column est un widget de mise en page qui aligne ses enfants verticalement (du haut vers le bas).

3
Ce qui s'affiche (1 pt) ?

L'écran affiche les widgets enfants de Column disposés verticalement dans un Scaffold avec une AppBar en haut. Le contenu précis dépend du code fourni dans le sujet.
